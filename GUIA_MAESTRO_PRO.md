# 🔱 CRM Maestro CREAR LIMA - Guía de Comandos Pro

## 1. Gestión de Reportes (WhatsApp)
- **Modo:** Sala de Guerra.
- **Lógica:** Pega el texto de las CCs. El sistema detectará el avance incremental (Delta).
- **Importante:** Estos reportes miden "Gestión de Llamadas" (Promesas), no asistencia real.

## 2. Auditoría de Desempeño
- **Modo:** Reporte de Desempeño.
- **Acción:** Pulsa "Sincronizar Data de la Web Ahora".
- **Foco:** Revisa las listas de "Sin Contestar" y "Sin Gestionar" con nombres propios.

## 3. Consola de Persecución
- **Ubicación:** Sidebar Izquierdo.
- **Lógica:** Solo muestra personas con DNI inválido y que **CERO ASISTENCIA** en C1.
- **Botón:** Envía el WhatsApp y el sistema lo ocultará automáticamente por hoy (Descarte).

## 4. Unificación de Identidad
- **Modo:** Centro de Integridad.
- **Acción:** Pulsa "Ejecutar Unificación Inteligente".
- **Resultado:** Deduplica la base y vincula graduados (Sello de Oro).

## 5. Bot Minero (DNI)
- El bot corre en segundo plano. No cierres la ventana negra si quieres que siga minando.
- Los resultados se guardan en `Mineria_DNIs.xlsx` y son inmortales.
